# Patient Zero

A 10 day long game jam game with the theme of "Just get started".
Our approach was making a zombie apocalypse game where the player is patient zero and has to start a zombie apocalypse.

## Goals

Make a fun and enjoyable game.
Get a good placing.
Get a web build working.


## Achievements

Reached top 8% in "fun" category out of 1150 submissions.
Reached top 14% in total score out of 1150 submissions.
Got the web build working.
