# Tune-Up Racing

A 2d top down racing game with a deep customization system. Also my first actual game I have developed.

## Goals

Get better at coding (I was still very new to coding at the time).
Become familiar with Github.
Publish game on itch.io.
Get feedback from players and improve game according to feedback.

## Achievements

Got a good understanding of important concepts in software development and why they are so important.
Improved my understanding of programming.
Got familiar with using Git.
Managed to get game published on itch.io.
Held playtests, fixed issues and added missing features according to player feedback.
Got the self discipline needed to work on a single project for 3 years in a row.
